### Install

```bash
npm i banana-converter
```

<!-- PROJECT SETUP -->

## Usage

```typescript
import { convert } from "banana-converter";

convert(1.5, "banana", "cm"); // 26.67
convert(24, "yard", "banana"); // 123.432
convert(60, "kilometer", "banana"); // 337078.651685
```
